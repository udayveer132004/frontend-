# AI Job Assistant

A comprehensive job search assistant with 6 key features:

## Features

1. **Resume Upload** - Upload and parse resumes (PDF, DOC, DOCX)
2. **ATS Score** - Score resume against job descriptions
3. **Resume Chat** - RAG-based Q&A about your resume
4. **Job Search** - Discover and match jobs
5. **Dashboard** - Track application status
6. **Mock Interview** - Voice-based interview practice

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

The app will open at http://localhost:3000

## Note

All backend responses are mocked for demonstration purposes.
